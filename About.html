Welcome to Recipes, your go-to platform for exploring and enjoying delicious recipes!

This website is part of my ITI Project, designed to showcase my skills in web development and user interface design.

Through this project, I focused on:

  • Applying web development technologies such as HTML, CSS, and JavaScript
  • Creating a responsive and visually appealing interface for browsing recipes

  • Combining creativity and technical skills to deliver a functional, elegant, and intuitive website

This project reflects my passion for coding, design, and culinary exploration, as well as my commitment to learning, improving my skills, and developing practical experience in web development.
